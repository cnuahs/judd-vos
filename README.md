# judd-vos

Judd-Vos-modified CIE 2-deg colour matching functions (1978)[^1]

## Data key

Columns: wavelength (nm), $\bar{x}^{\prime}$, $\bar{y}^{\prime}$, $\bar{z}^{\prime}$

## Notes

In 1978, Vos made additional corrections to Judd's (1951) revision of the [CIE 1931 2-deg](https://cie.co.at/datatable/cie-1931-colour-matching-functions-2-degree-observer) color matching functions (CIE, 1932) and incorporated the infra-red color reversal described by Brindley (1955).

From Table 1 of Vos (1978). See also Table I (5.5.2) of Wyszecki and Stiles (1982).
